# Задача: Добавить фильтрацию задач: все / активные / выполненные.

Требования:

- Три кнопки для переключения фильтра
- 'Все' — показывает все задачи
- 'Активные' — только невыполненные
- 'Выполненные' — только выполненные
- Активный фильтр визуально выделен

Подсказка: Используйте computed для отфильтрованного массива.

## Критерий успеха: Переключение фильтров корректно фильтрует список.

Время: 30-40 минут
