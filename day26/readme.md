# Задача: Создать страницу корзины с возможностью удаления.

Требования к pages/cart.vue:

- Если корзина пуста: сообщение 'Корзина пуста' + ссылка на главную
- Список товаров: изображение, название, количество, цена за шт, сумма
- Кнопка 'Удалить' для каждого товара
- Итоговая сумма внизу
- Кнопка 'Оформить заказ' (пока alert: 'Заказ оформлен!')

## Критерий успеха: Можно посмотреть корзину, удалить товары, увидеть сумму.

Время: 40-50 минут
