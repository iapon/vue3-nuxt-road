# Задача: Создать composable для логики счётчика.

Требования к composables/useCounter.js:

- Возвращает: count (ref), increment(), decrement(), reset()
- Опциональный параметр: initialValue (по умолчанию 0)
- Опциональный параметр: min и max (границы значений)

Требования к App.vue:

- Создайте два счётчика с разными настройками
- Первый: обычный (0-100)
- Второй: с начальным значением 10 и границами 5-15

## Критерий успеха: Два независимых счётчика с разными настройками.

Время: 35-45 минут
