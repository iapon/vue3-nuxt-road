# Задача: Создать палитру цветов с предпросмотром.

Требования к компоненту ColorPicker.vue:

- Массив из 6-8 цветов на выбор
- Цвета отображаются как цветные кнопки/круги
- При клике — выбранный цвет выделяется рамкой
- Квадрат 100x100px залит выбранным цветом
- Текст: 'Выбран: [название цвета]'

Подсказка: Используйте @click и :style для динамических стилей.

## Критерий успеха: Клик меняет цвет квадрата и название.

Время: 35-45 минут
