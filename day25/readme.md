# Задача: Создать composable для корзины и счётчик в шапке.

Требования к composables/useCart.js:

- items — массив товаров в корзине
- addItem(product) — добавить товар (если есть — увеличить quantity)
- removeItem(productId) — удалить товар
- total — вычисляемая сумма всех товаров
- count — общее количество товаров

Требования к layouts/default.vue:

- Добавьте иконку корзины в шапку (ссылка на /cart)
- Покажите badge с количеством товаров (count)

Требования к pages/product/[id].vue:

- Кнопка 'В корзину' вызывает addItem(product)
- При добавлении — краткое уведомление 'Добавлено!'

## Критерий успеха: Товары добавляются, счётчик в шапке обновляется.

Время: 50-60 минут
