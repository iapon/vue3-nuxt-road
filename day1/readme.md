## День 1: Создание проекта и компонент Greeting

*Задача: Создать проект Vue 3 через Vite и компонент приветствия.*

Требования к компоненту Greeting.vue:

• Поле ввода для имени пользователя

• Текст приветствия появляется только когда имя введено

• Если поле пустое — показать сообщение 'Введите ваше имя'

• Имя должно отображаться в приветствии

Подсказка: Используйте v-model для привязки input и v-if/v-else для условного отображения.

**Критерий успеха: Вводите имя — видите 'Привет, [имя]!'. Стираете — видите приглашение.**

Время: 30-45 минут
