# Задача: Создать секундомер, который сохраняется в localStorage.
Требования к компоненту Timer.vue:

- Отображение времени в формате HH:MM:SS
- Кнопки: Старт / Стоп / Сброс
- При обновлении страницы время сохраняется
- Кнопка Старт неактивна во время работы, Стоп — неактивна при паузе

Подсказка: Используйте watch для сохранения в localStorage при изменении секунд.

## ритерий успеха: Таймер работает, сохраняется после перезагрузки.

Время: 45-55 минут
