# Задача: Создать карточку пользователя с props для данных.

Требования к компоненту UserCard.vue:

- Props: name (обязательный), email (по умолчанию 'email@example.com'), role (по умолчанию 'user')
- Если avatar не передан — показать круг с первой буквой имени
- Роль отображается в цветном бейдже
- Карточка имеет рамку и отступы

Требования к App.vue:

- Отобразите 3 карточки с разными данными
- Одна карточка только с name (проверка дефолтных значений)

Подсказка: Используйте defineProps с валидацией типов.

## Критерий успеха: Карточки отображают разные данные, дефолтные значения работают.

Время: 35-45 минут
