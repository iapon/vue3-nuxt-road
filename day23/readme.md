# Задача: Загрузить товары категории из FakeStore API.

Требования к pages/category/[id].vue:

- Определите категорию по id (1=electronics, 2=jewelery, 3=men's clothing, 4=women's clothing)
- Загрузите товары: https://fakestoreapi.com/products/category/[category]
- Отобразите товары в сетке (3-4 колонки)

Требования к components/ProductCard.vue:

- Prop: product (объект товара)
- Изображение, название, цена
- Ссылка 'Подробнее' на /product/[id]

## Критерий успеха: Товары загружаются, отображаются в сетке.

Время: 50-60 минут
