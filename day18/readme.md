# Задача: Загрузить посты из JSONPlaceholder API.

Требования к pages/blog/index.vue:

- Используйте useFetch для загрузки с https://jsonplaceholder.typicode.com/posts?_limit=10
- Показывайте 'Загрузка...' пока данные грузятся
- Отобразите 10 постов: title и body (обрезанный до 100 символов)

Требования к pages/blog/[id].vue:

- Загрузите конкретный пост по id из API
- Отобразите title и полный body

## Критерий успеха: Посты загружаются из API, детальная страница работает.

Время: 40-50 минут
