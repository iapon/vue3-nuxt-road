# Задача: Создать страницу с виджетами всех компонентов недели.

Требования:

- Сетка 2x2 с виджетами: Counter, TodoList, ColorPicker, Timer
- Каждый виджет в карточке с заголовком и рамкой
- Импортируйте все созданные компоненты в App.vue
- Стилизуйте с помощью CSS Grid или Flexbox

## Критерий успеха: 4 виджета отображаются в сетке, каждый работает.

Время: 30-40 минут
