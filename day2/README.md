## День 2: Компонент Counter

**Задача: Создать счётчик с кнопками увеличения и уменьшения.**

Требования к компоненту Counter.vue:

• Отображение текущего числа (начинается с 0)

• Кнопка '+' увеличивает на 1

• Кнопка '-' уменьшает на 1

• Кнопка 'Сброс' возвращает к 0

• Число не может быть меньше 0 (блокировать кнопку '-')

Подсказка: Используйте ref() для реактивной переменной и @click для обработчиков.

### Критерий успеха: Счётчик работает, не уходит в минус, сбрасывается.
Время: 30-40 минут
